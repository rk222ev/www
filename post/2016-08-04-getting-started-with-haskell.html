Getting started with Haskell

This guide will only show how to get going with Haskell using OS X and will not be very detailed.

Installation

The things needed to get started are The Glasgow Haskell Compiler (ghc) and the package manager Cabal. 

$ brew update
$ brew install ghc cabal-install

Running "Hello World!"

Create a file named something like hello-world.hs. Make it look like: 

-- hello-world.hs

main = putStrLn "HelloWorld"

Run your program by doing using the command `runhaskell`. 

$ runhaskell hello-world.hs
Hello World

Using community packages

Here I'll show how to install and use The HTTP package to make a request to google and print the response body to stdout.

Haskell has a community driven "central package archive of open source software": Hackage.

Unfortunately the names of the packages you are supposed to use when installing doesn't directly translate over to the import statement you use in your code. For now lets just accept that the package Network.HTTP is installed with Cabal using the name http and Network.HTTP.Conduit is http-conduit. Exactly how this works is a bit unclear at the moment from just looking at these two packages but examining some more packages should make the picture clearer.

Anyhow. To install the http package from Hackage simply use Cabal: 

$ cabal install http

Create a new file named something like google.hs and enter the following: 

-- google.hs

import Network.HTTP

main :: IO ()
main = do
  body <- getBody "http://www.google.se"
  putStr body

getBody :: String -> IO (String)
getBody x = getResponseBody =<< simpleHTTP (getRequest x)

Once again run the program with: 

$ runhaskell google.hs
